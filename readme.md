## アナリティクスの使い方
- 参考サイト

- 導入する方法
    - STEP1　プロパティの作成
    - STEP2　設定
    - STEP3　トラフィックコードの設
- Google アナリティクス 4（GA4）のイベントを使う
    - 主なイベント
        - page_view：ページが読み込まれると計測される
        - first_visit：ウェブサイトにユーザーが初めてアクセスしたときに計測される
        - session_start：ユーザーがウェブサイトを利用してセッションが開始されたときにのみ計測される
